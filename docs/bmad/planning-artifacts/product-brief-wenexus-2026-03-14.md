# Product Brief: wenexus

## Executive Summary

WeNexus 让你看到事情的全貌。

在一个物质日益丰裕、精神却日趋撕裂的时代，人们面对争议话题时，看到的往往只是"我方的真相"。WeNexus 围绕人类社会的经典争议问题（彩礼、内卷、AI 取代人类、阶层固化……），通过 AI 专家圆桌讨论，结构化地呈现不同立场的观点碰撞，让用户自己拼出全貌——而不是被任何单一视角的"全面解析"所局限。

产品采用双层架构：表层是小红书式的观点卡片信息流，每张卡片鞭辟入里地击中用户的认知盲区，制造"啊，原来如此"的认知惊喜；深层是 AI 专家圆桌讨论，提供深度互动和个性化内容产出（文章、脚本、决策清单等）。

商业上，用户因为内容精彩而来，因为认知提升而留。通过内容消费驱动增长，通过深度服务（私人圆桌、专业产出）实现付费转化。以 AI Agent 在各内容平台运营 WeNexus 产出内容导流，实现低成本冷启动。

WeNexus 的核心信念：复杂问题不存在简单答案。当哲学家和社会学家看到了现代人类的认知困境，破局之道需要企业家的产品来提供。

## Core Vision

### Problem Statement

全球社会正经历一场深层的理解危机：国与国之间不再相信全球化，国内政治极化加剧，不同阶级与地域的人们之间的理解鸿沟越来越大。物质生活的改善非但没有弥合分歧，反而因为信息茧房和算法推荐，让人们更加固守"我方的真相"。

项飙说，"严肃性和深度是解压的最好方式"，但当前的内容平台恰恰在做相反的事——用情绪化、碎片化、极化的内容收割注意力，让"有意义的对话在无效中消解了所有的意义"。

人们不缺信息，缺的是全貌。

### Problem Impact

- **个人层面**：用户在社交媒体上越刷越焦虑，对争议话题只能看到局部，产生"我是对的、对方是蠢的"的认知幻觉
- **社会层面**：公共讨论质量持续下降，"各说各话"成为常态，建设性对话越来越稀缺
- **创作者层面**：内容创作者需要多角度素材，但自行搜集整理耗时且视角有限；AI 问答工具只给单一答案，缺乏观点碰撞
- **决策者层面**：面对复杂决策需要多方意见模拟，现有工具无法提供结构化的多元视角分析

### Why Existing Solutions Fall Short

| 产品 | 承诺 | 实际交付 | WeNexus 的差异 |
|------|------|---------|---------------|
| 知乎/B站 | "全面解析" | 单一作者视角的全貌（作者的盲区即用户的盲区） | 结构化多视角呈现，用户自己判断、自己拼图 |
| ChatGPT/Kimi | "客观回答" | 刻意中立的折中答案（回避真正的分歧点） | 多专家辩论，直面冲突，展示"为什么有人会这样想" |
| 微博/Twitter | 时效热点 | 算法鼓励极化，评论区情绪化对骂 | 聚焦经典争议问题，AI 确保每种立场都有高质量表达 |
| 播客（小宇宙等） | 深度对话 | 90 分钟只获得 3-4 个观点，时间成本极高 | 30 秒一张观点卡片，同等信息密度，碎片时间可消费 |
| 得到/知识付费 | 知识提升 | 单向输出，学习是反人性的 | "看辩论"比"听课"有趣，娱乐性与深度兼具 |

### Proposed Solution

WeNexus 通过"双层内容架构"解决这个问题：

**第一层：观点卡片信息流（获客层）**

- 小红书式的精美图文卡片，每张卡片鞭辟入里地呈现一个专家视角
- 聚焦经典争议问题，内容具有持久价值，不会因新闻过时而失效
- 设计标准：一句让你停下来的话 + 一个你没想到的角度 + 一个想看更多的理由
- 适合碎片化消费和社交传播，低门槛吸引用户

**第二层：AI 专家圆桌讨论（深度层）**

- 求真者（Fact Checker）先做事实打底，确保基于事实辩论
- 多位 AI 专家进行线程式对话，明确引用和回应彼此观点
- 用户可"挂机围观"或"接管主持"，控制讨论方向
- 讨论成果一键转化为文章、脚本、决策清单等可复用产出

**分发策略：**

- 以 AI Agent 在各内容平台（小红书、微博、公众号等）运营 WeNexus 产出内容
- 用优质内容自然导流回平台，实现低成本冷启动

### Key Differentiators

1. **全貌 vs 局部**：不是一个人讲"来龙去脉"，而是让不同立场的专家各自讲自己看到的部分，用户自己拼出全貌。比任何单一视角的"全面解析"都更可信
2. **经典问题 vs 时效新闻**：不追热点，聚焦有持久价值的争议话题。新闻会过期，但"阶层固化""教育公平""AI 取代人类"这类问题不会。每条内容都是资产，不是消耗品
3. **认知惊喜 vs 情绪刺激**：现有平台制造愤怒以提升参与度，WeNexus 制造"啊，原来如此"的好奇心以提升理解度。用户驱动力是"我不想被蒙在鼓里"（自利），而非"我要理解对方"（利他）
4. **内容消费 vs 说教平台**：使命是促进理解，但产品体验是内容消费。用户因为内容精彩而来，认知提升是自然发生的副产品
5. **AI Agent 分发**：用 Agent 在各平台运营内容导流，几乎零人力成本实现增长
6. **产出物即传播载体**：每张观点卡片、每篇生成文章自带品牌水印，用户分享即推广

## Target Users

### Primary Users

**用户层级一：觉醒中的旁观者（The Awakening Bystander）**

**代表人物**：任何年龄段、任何生活阶段的普通人

这不是一个特定人群，而是一种**状态**——当一个人面对某个争议话题时，他处于以下三种状态之一：

- **被洗脑但不自知**：已经接受了某一方的叙事，认为自己"想明白了"
- **不舒服但说不清**：隐约感觉哪里不对，但无法表达，也不知道去哪里找答案
- **搜过但只得到局部**：在知乎/B站看了"全面解析"，以为懂了，实际上只看到了作者的视角

**覆盖的生命议题谱：**

- 青年期：内卷、考研还是就业、租房还是买房
- 婚恋期：彩礼、婚前同居、原生家庭、生育选择
- 职场期：跳槽、向上管理、职场政治、35岁危机
- 中年期：中年危机、教育焦虑、阶层固化
- 宏观层面：两党对立、国际冲突、历史争议、AI取代人类、社会演化规律

**核心驱动力**：自利——"我不想被蒙在鼓里"，而非利他——"我要理解对方"。本质上是消除内容平台刻意制造的信息不对称。

**竞品行为**：去知乎搜、问ChatGPT、刷微博评论区——但更多时候，用户根本不会主动搜索，因为他甚至不知道自己只看到了局部。

---

**用户层级二：积极的连接者（The Proactive Connector）**

**代表人物**：王沛然

在"觉醒中的旁观者"看到全貌之后，一部分人会产生行动的欲望——她不仅想搞明白，还想**用这些理解去做点什么**：修复一段关系、做一个更好的决策、在一次争论中真正说服对方。

**与层级一的区别**：层级一的用户消费完观点卡片会感到满足（"原来如此"）；层级二的用户会进入圆桌讨论、主持对话、生成决策清单或沟通策略。

**转化路径**：层级一 → 被内容持续打动 → 自然升级为层级二。这不是两类不同的人，而是**同一个人在不同阶段**。

### Secondary Users（后期规划）

**内容创作者**：当 WeNexus 自身内容生产能力成熟后，将多余产能和工具能力开放给需要多角度素材的视频/文章创作者。

**广义知识需求者**：所有需要完整信息地图的人、希望深入讨论感兴趣话题的人（教育工作者、媒体人、企业管理者等）——这些是产品成熟后自然拓展的用户群。

### User Journey

**第一触点：被击中（平台外）**

用户不是主动搜索找到 WeNexus 的。他在刷小红书/微博/公众号时，看到一张 WeNexus AI Agent 发布的观点卡片——一句让他停下来的话、一个他没想到的角度。第一反应是："等等，这个说法我怎么没想到？"

**第二步：发现全貌（进入 WeNexus）**

好奇心驱动他点进来，看到同一个话题下多位专家的不同视角卡片。不是一个人的"全面解析"，而是结构化的观点碰撞。他意识到："原来这件事还有这么多面。"

**Aha Moment**：第一次看到自己坚信的观点被一个意料之外的角度刺穿。不是被说服了，而是意识到**自己之前只看到了一面**。

**第三步：越看越深（留存）**

他开始浏览其他话题——那些他以为自己已经想明白的事情。每一个话题都给他新的认知惊喜。WeNexus 成为他面对争议话题时的第一反应："先去看看全貌。"

**第四步（进阶用户）：从围观到参与**

一部分用户会进入 AI 专家圆桌讨论，从"挂机围观"到"接管主持"，把讨论引向自己关心的方向，最终生成可用的产出物（文章、决策清单、沟通策略）。

## Success Metrics

### 用户成功指标

**北极星指标：认知惊喜率**

- 用户看到观点卡片后产生"想看更多"行为的比例（点击进入话题详情 / 浏览下一张卡片）
- 这是 WeNexus 核心价值的直接衡量——内容是否真的击中了认知盲区

**用户价值行为（按漏斗排序）：**

| 行为 | 衡量的价值 | 指标 |
|------|-----------|------|
| 浏览观点卡片 | 内容吸引力 | 单次会话浏览卡片数、停留时长 |
| 分享观点卡片 | 内容传播力 | 分享率、分享后回流率 |
| 浏览同话题多视角 | 全貌价值 | 单话题多卡片浏览率 |
| 进入圆桌讨论 | 深度需求 | 卡片→圆桌转化率 |
| 生成产出物 | 行动价值 | 产出物生成次数、下载/分享次数 |

**Aha Moment 定义：** 用户第一次在同一话题下浏览 3+ 个不同立场的观点卡片——这是"发现全貌"的最小行为信号。

### Business Objectives

**Phase 1：技术验证（0-3 个月）**

目标：跑通"AI 内容生产 → 外部平台分发 → 用户回流"的完整闭环。

| 验证点 | 成功标准 |
|--------|---------|
| 内容生产 | AI 能自动生成高质量观点卡片（人工审核通过率 > 80%） |
| 圆桌讨论 | AI 专家能进行有质量的多角度辩论，产出结构化内容 |
| 外部分发 | AI Agent 能在至少 1 个平台（小红书/公众号）稳定运营 |
| 回流链路 | 外部平台→WeNexus 的导流路径技术上可行 |

**这个阶段不追求用户量级，核心是验证技术可行性。**

**Phase 2：用户增长与付费验证（3-12 个月）**

目标：稳定的用户增长和付费转化。

| 指标 | 方向 |
|------|------|
| 用户增长 | 外部分发带来的自然增长趋势稳定向上 |
| 留存 | 次周留存率作为内容价值的核心验证 |
| 付费转化 | 免费→付费会员的转化率，验证商业模式可行 |
| 内容规模 | 话题库覆盖的经典问题数量持续增长 |

### Key Performance Indicators

**商业模式：会员积分制**

| 层级 | 权限 | 商业逻辑 |
|------|------|---------|
| 免费用户 | 浏览观点卡片信息流 | 获客层：用内容质量吸引用户 |
| 付费会员 | 圆桌讨论 + 产出物生成 | 变现层：深度服务驱动付费 |

**核心 KPI 框架：**

- **获客效率**：单条外部平台内容带来的 WeNexus 访问量
- **内容质量**：观点卡片的浏览完成率、分享率
- **转化漏斗**：免费浏览 → 注册 → 进入圆桌 → 付费会员
- **付费价值**：付费用户的圆桌使用频次、产出物生成量、续费率

## MVP Scope

### Core Features

**1. AI 内容生产引擎**

MVP 的技术核心。围绕经典争议话题，AI 自动生成结构化多视角内容：

- 话题拆解：将一个经典问题拆解为多个争议维度
- 专家视角生成：为每个维度生成不同立场的专家观点
- 观点卡片渲染：将专家观点转化为小红书式精美图文卡片（AIGC 直出 JSX）
- 质量验证：前 50 个话题人工审核打磨，验证 AI 生成质量达标后，批量扩展至 10000+
- 生产链路：离线批量生产 + 增量在线生成

**2. 观点卡片信息流**

用户端的核心消费体验：

- 话题维度的观点卡片瀑布流/信息流
- 同一话题下多立场卡片的结构化浏览
- 卡片分享功能（自带品牌水印，分享即传播）

**3. AI 专家圆桌讨论**

深度层产品体验：

- 求真者（Fact Checker）事实打底
- 多位 AI 专家线程式辩论，明确引用和回应彼此
- 用户"挂机围观"与"接管主持"两种模式
- 讨论成果一键转化为文章、决策清单等产出物

**4. 外部平台 Agent 分发**

增长引擎：

- AI Agent 在至少 1 个内容平台（小红书/公众号）自动运营
- 发布 WeNexus 产出的观点卡片内容
- 导流链路：外部平台 → WeNexus 站内

**5. 会员付费系统**

商业验证闭环：

- 免费层：浏览观点卡片信息流
- 付费层：使用圆桌讨论 + 产出物生成
- 会员积分制，验证付费转化可行性

### Out of Scope for MVP

| 功能 | 不做的原因 |
|------|-----------|
| 用户社交功能 | 核心价值是内容消费，非社交互动 |
| 评论系统 | 避免引入低质量 UGC，与平台调性冲突 |
| 内容创作者工具 | 待自身内容生产能力成熟后再开放 |
| 多语言内容 | 一开始面向全球但只生产中文内容，贴近中国市场 |
| 个性化推荐算法 | Phase 1 内容规模有限，先以话题分类组织 |

### MVP Success Criteria

**Phase 1 通过标准（3 个月）：**

| 验证维度 | 通过条件 |
|---------|---------|
| 内容生产 | 50 个话题人工审核通过率 > 80%，具备批量扩展至 10000 的技术基础 |
| 产品体验 | 用户能完成"浏览卡片→进入圆桌→生成产出物"的完整链路 |
| 分发导流 | 至少 1 个外部平台 Agent 稳定运营，能产生回流 |
| 付费验证 | 付费系统可用，能观察到自然付费转化行为 |

**决策点**：Phase 1 通过后进入 Phase 2（用户增长与规模化），否则回溯调整内容策略或产品形态。

### Future Vision

**短期（1-2 年）：多元视角内容平台**

- 10000+ 经典话题全覆盖，AI 批量生产+增量在线生成
- 多平台 Agent 矩阵分发（小红书、微博、公众号、B站等）
- 付费会员稳定增长，商业模式跑通
- 多语言内容扩展，服务全球中文用户

**中期（2-3 年）：全球认知升级信息中心**

- 多语言内容覆盖，真正服务全球用户
- 成为争议话题"看全貌"的第一选择——用户面对争议问题时的第一反应是"先去 WeNexus 看看"
- 开放内容创作者工具，将多余产能赋能创作者生态

**长期愿景：Nexus 人的社区与组织**

- 从内容平台进化为认知共同体——"Nexus 人"是那些主动追求全貌、拒绝被单一叙事裹挟的人
- 构建 Nexus 人的社区组织，从线上认知到线下连接
- 全球人类认知升级的基础设施
